My project is Card Vault.
This project is a secure payment authorization system designed to validate card transactions received from POS devices using industry-grade cryptography and PCI-DSS compliant components. When a customer taps or swipes a card on a POS machine, the card data is captured and the PIN is encrypted using AES-256 within the POS secure PIN pad. The encrypted PIN, along with transaction details, is routed to the Card Authentication Service, which coordinates the flow using a Ticketing Service for secure vault access. The request then moves through an HSM Proxy Service that interacts with PINsafe and TrackSafe vaults to fetch encrypted PIN blocks and track data. These encrypted elements are forwarded to the HSM Service (Thales), which performs PIN verification, cryptogram validation, and PCI-compliant security checks. After verification, the response flows back through the HSM Proxy and Card Authentication Service to the POS terminal, completing the payment approved/declined flow securely and efficiently.
